# 🔗 URL Shortener - Task 1

## 📌 Project Overview
A simple URL shortener built with **Flask** and **SQLite** as part of the **CodeAlpha Backend Development Internship**.

## ✨ Features
- ✅ Shorten long URLs
- ✅ Generate unique short codes
- ✅ Redirect to original URLs
- ✅ Store mappings in SQLite database
- ✅ Error handling for invalid short codes
- ✅ Simple web interface

## 🛠️ Technologies Used
- **Python** - Backend logic
- **Flask** - Web framework
- **SQLAlchemy** - ORM for database
- **SQLite** - Database
